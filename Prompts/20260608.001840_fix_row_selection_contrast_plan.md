# User Prompt
when row is selected in "Active downloads", the row font color is indistinct from the background color of the list.

# Goals
Fix the font color contrast of selected rows in the "Active downloads" table (and all table views) so that text is highly visible and clear when a row is selected.

## Proposed Changes

### Styling (CSS)

#### [MODIFY] [index.css](file:///x:/Projects_X/0_Active/1_Java_Active/Torrent_Share_Java_GITHUB/src/main/resources/index.css)
- Add a CSS rule `.table-view .table-row-cell:filled:selected .table-cell` to set the text color of the cells in selected rows to `-text-primary`. This ensures that standard table cells use the correct thematic text color rather than falling back to the default JavaFX selection text color (white), which has very poor contrast against our custom light/translucent selected row background (`-row-selected`).

## Verification Plan

### Manual Verification
- Compile and run the application.
- Navigate to the downloads overview/table.
- Select a row in the table.
- Verify that the text color of all columns remains clearly visible (`-text-primary`) rather than turning white/indistinct.
