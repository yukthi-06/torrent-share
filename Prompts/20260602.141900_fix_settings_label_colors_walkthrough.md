# Walkthrough - Fix Settings Tab Label Text Color

## Problem
Labels in SettingsTab were invisible (white text on white/light background). The ShareTab used `-fx-text-fill: -fx-text-base-color` as inline styles on labels — SettingsTab had no inline color at all, relying on CSS inheritance which was failing.

## Changes Made

### [`SettingsTab.java`](file:///x:/Projects_X/0_Active/1_Java_Active/Torrent_Share_Java_GITHUB/src/main/java/com/vypeensoft/torrentshare/ui/SettingsTab.java)
Added `-fx-text-fill: -fx-text-base-color` inline style to all labels:
- Section titles: `genTitle`, `trackTitle`
- Grid row labels (field names)
- Hint/tooltip labels

### [`index.css`](file:///x:/Projects_X/0_Active/1_Java_Active/Torrent_Share_Java_GITHUB/src/main/resources/index.css)
- Added global `.label` rule: `-fx-text-fill: -text-primary`
- Added `ListView` dark theme styling
- Fixed `.scroll-pane > .viewport` to `transparent` to prevent white-wash on dark backgrounds
