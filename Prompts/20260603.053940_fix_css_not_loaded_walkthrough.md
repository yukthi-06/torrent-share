# Walkthrough - Fix CSS Not Loading (index.css Never Registered with Scene)

## Root Cause
`index.css` was never added to the JavaFX `Scene`'s stylesheet list. All CSS changes made to `index.css` had absolutely zero effect on the running app because the stylesheet was never loaded.

In `MainWindow.java`, the Scene was created without loading the stylesheet:
```java
// BEFORE — css never loaded
Scene scene = new Scene(root, defaultWidth, defaultHeight);
```

## Fix Applied

### [`MainWindow.java`](file:///x:/Projects_X/0_Active/1_Java_Active/Torrent_Share_Java_GITHUB/src/main/java/com/vypeensoft/torrentshare/ui/MainWindow.java)

Added stylesheet registration after scene creation:
```java
Scene scene = new Scene(root, defaultWidth, defaultHeight);
String css = getClass().getResource("/index.css").toExternalForm();
scene.getStylesheets().add(css);
```

This loads `index.css` from the classpath resources and applies it to the entire scene tree — all nodes, all tabs, all buttons.
