# Implementation Plan - Remove Dark Theme & Restore System Default Modena Theme

This plan outlines the changes required to remove all custom dark-theme styling, glassmorphism CSS stylesheets, and hardcoded dark/light colors from the TorrentShare application to fully restore the JavaFX system default Modena (light) theme.

## User Review Required

> [!IMPORTANT]
> Because the JavaFX system default theme is light, all hardcoded white (`Color.WHITE`) and light-gray (`#9ca3af`, `#6b7280`) text fills and styles must be removed or adjusted so that the text is automatically styled by the default system color palette (black/dark-gray text on light background).
