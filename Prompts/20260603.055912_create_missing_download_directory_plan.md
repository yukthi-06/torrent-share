When adding a magnet link, if there folder is missing, it shows an error "Please select a valid, existing download directory".

# Implementation Plan - Create Missing Download Directory on Magnet Add

## Goal
Improve user experience by automatically attempting to create non-existing folders specified in the save path field of the "Add Magnet Link" dialog, rather than rejecting the input.

## Proposed Changes

### Add Directory Creation Logic
#### [MODIFY] [AddMagnetDialog.java](file:///x:/Projects_X/0_Active/1_Java_Active/Torrent_Share_Java_GITHUB/src/main/java/com/vypeensoft/torrentshare/ui/AddMagnetDialog.java)
- In `handleAdd()`, if `savePath` is not empty:
  - Check if the target `File` directory exists.
  - If it does not exist, call `dir.mkdirs()` to create all necessary parent directories.
  - If creation fails, show a descriptive error: `"Failed to create download directory."`.
  - If it exists but is not a directory (e.g. is a file), show: `"Specified path is not a directory."`.
