# Implementation Plan - Fix Share Tab Color Visibility & Contrast

This plan outlines the changes required to ensure that all text elements inside the "Share Files" tab adapt dynamically to the system background (using standard JavaFX system-default CSS color tokens), guaranteeing perfect legibility on any light or dark background configuration.

## User Review Required

> [!NOTE]
> When stylesheet themes are disabled, standard Label elements may fail to compute appropriate contrast against parent background variations if their styling is not explicitly linked.
>
> We will resolve this by:
> 1. Setting `setStyle("-fx-text-fill: -fx-text-background-color;");` on all key text elements in the Share tab. The `-fx-text-background-color` token is a special JavaFX Modena variable that dynamically resolves to dark-gray/black on light backgrounds and white on dark backgrounds.
> 2. Setting standard Modena container borders and background styling on the `infoPanel` VBox to ensure it renders as a clean visual section.
