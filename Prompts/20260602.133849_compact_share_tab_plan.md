# Implementation Plan - Compact Share Tab Layout

This plan outlines the changes required to reduce padding, margins, row gaps, text sizes, and text area heights in the "Share Files" tab to make the entire layout extremely compact and minimize required screen vertical space.

## User Review Required

> [!NOTE]
> We will decrease:
> 1. General tab paddings and margins by ~50%.
> 2. Drag-and-drop container minimum height and padding from 200px/48px to 120px/20px.
> 3. Vertical layout spacing gaps (`VBox` spacing and `GridPane` row gaps) from 20px/12px to 10px/6px.
> 4. Magnet text field height from 4 rows to 2 rows.
