# Walkthrough - Fix JavaFX Alert Dialog Text Clipping

We have successfully resolved the text clipping and cropping issues inside the JavaFX Alert warning and error dialogs by setting the DialogPane's minimum height constraint to `Region.USE_PREF_SIZE`.

## Changes Made

### UI Components

#### [ShareTab.java](file:///x:/Projects_X/0_Active/1_Java_Active/Torrent_Share_Java_GITHUB/src/main/java/com/vypeensoft/torrentshare/ui/ShareTab.java)
- Updated both `showError(String title, String details)` and `showWarning(String title, String details)` alert helper methods to invoke:
  `alert.getDialogPane().setMinHeight(javafx.scene.layout.Region.USE_PREF_SIZE);`
  This instructs JavaFX's layout engine to dynamically scale the dialog box heights based on the preferred dimensions of their multiline content strings.

## Verification Results

### Dialog Legibility
- Dropping multiple files triggers the warning popup which now displays **100% of the message text** (including the instruction to package in a single folder) cleanly without any truncation or cutoff at the bottom.
- Any error alerts spawned during app lifecycle will also dynamically scale to prevent message clipping.
