# Walkthrough - Add Scrollability to Share Files Tab

We have successfully wrapped the "Share Files" tab layout inside a transparent, fit-to-width JavaFX `ScrollPane` container. This completely eliminates UI element clipping and enables natural vertical scrolling when the application window is resized smaller than the content height.

## Changes Made

### UI Components

#### [ShareTab.java](file:///x:/Projects_X/0_Active/1_Java_Active/Torrent_Share_Java_GITHUB/src/main/java/com/vypeensoft/torrentshare/ui/ShareTab.java)
- Imported `javafx.scene.Parent`.
- Declared class-level `ScrollPane scrollPane;` field.
- In `buildUI()`, instantiated `scrollPane = new ScrollPane(root);` wrapping the core `VBox` root container holding the drag zone and metadata form panel.
- Configured the scrollpane to stretch its layout horizontally with `scrollPane.setFitToWidth(true)`.
- Set elegant transparent background and border styling to avoid double-borders or dark scroll containers:
  `-fx-background-color: transparent; -fx-background: transparent;`
- Refactored `getContent()` signature and body to return the new scrollable `Parent` instead of the raw `VBox`.
