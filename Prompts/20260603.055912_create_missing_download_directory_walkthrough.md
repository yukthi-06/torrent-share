# Walkthrough - Create Missing Download Directory on Magnet Add

## Goal
Improve usability by automatically creating non-existent save paths instead of rejecting them with an error.

## Changes Made

### [`AddMagnetDialog.java`](file:///x:/Projects_X/0_Active/1_Java_Active/Torrent_Share_Java_GITHUB/src/main/java/com/vypeensoft/torrentshare/ui/AddMagnetDialog.java)
Modified `handleAdd()` to dynamically create folders if the target save directory does not yet exist:
* Checks if `savePath` is blank.
* Checks if the `File` path exists. If not, it runs `dir.mkdirs()` to create all missing directories in the tree.
* Shows an error message if the directory creation fails or if the path exists but points to a file rather than a directory.
