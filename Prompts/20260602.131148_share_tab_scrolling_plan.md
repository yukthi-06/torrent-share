# Implementation Plan - Add Scrollability to Share Files Tab

This plan outlines the changes required to wrap the "Share Files" tab panel in a JavaFX `ScrollPane` container, ensuring the layout is fully scrollable and legible when resized below content dimensions.

## User Review Required

> [!NOTE]
> Currently, `ShareTab` returns its main `VBox` container directly. When resized, vertical dimensions cut off form metadata and export buttons.
>
> We will resolve this by wrapping the VBox in a `ScrollPane` configured with `setFitToWidth(true)` and transparent background styles, making it scroll vertically when needed.

## Open Questions

None.

## Proposed Changes

### UI Layer

#### [MODIFY] [ShareTab.java](file:///x:/Projects_X/0_Active/1_Java_Active/Torrent_Share_Java_GITHUB/src/main/java/com/vypeensoft/torrentshare/ui/ShareTab.java)
- Import `javafx.scene.Parent` and `javafx.scene.control.ScrollPane`.
- In `buildUI()`, instantiate `ScrollPane` wrapping the core `VBox` container.
- Configure the scrollpane with `setFitToWidth(true)` and transparent background and borders.
- Update `getContent()` to return `Parent` (the `ScrollPane` instance) instead of the raw `VBox`.
