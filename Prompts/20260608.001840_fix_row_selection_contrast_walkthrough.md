# User Prompt
when row is selected in "Active downloads", the row font color is indistinct from the background color of the list.

# Walkthrough of Changes

To address the low contrast of selected rows in the table views, we updated the application stylesheet.

### Changes Made

#### 1. TableView Selection Style Update
In [index.css](file:///x:/Projects_X/0_Active/1_Java_Active/Torrent_Share_Java_GITHUB/src/main/resources/index.css):
- Added the `.table-view .table-row-cell:filled:selected .table-cell` selector rule to set the `-fx-text-fill` of selected table cells to `-text-primary`.
- This overrides JavaFX's default behavior of converting cell text to white for selected rows, ensuring high legibility and contrast against our light/translucent custom selected row background (`-row-selected`).
